# Jax Learning Tracker

Jax Learning Tracker 是一个极简每日学习记录工具，用本地网页记录每天学了多久、学了什么、遇到什么问题，以及明天继续做什么。

## 当前版本功能

- 记录日期、学习时长、学习内容、视频学习到了什么、遇到的问题、明天下一步任务
- 可以勾选今日评分项：AI 编程、视频剪辑 / 素材整理、复盘笔记、认知视频
- 保存时会根据 4 个评分项和学习时长自动计算 0-100 分，并生成今日判断
- 按日期保存记录，每个日期只保留一条记录
- 同一天多次点击「保存学习记录」只会更新同一天记录，不会新增重复卡片
- 切换日期时，如果该日期已有记录，会自动加载到表单；如果没有记录，会清空内容并保留日期
- 学习时长可以为空；如果填写，必须是 0 或正数
- 只要日期存在就可以保存，学习时长、学习内容、遇到的问题、明天下一步任务都可以为空
- 清空表单后保存，会把当前日期记录更新为空内容
- 历史记录按日期倒序显示
- 每张历史卡片显示学习评分、判断、学习时长、学习内容、视频学习到了什么、遇到的问题和明天下一步任务
- 每张历史卡片可以加载到表单，也可以删除
- 支持导出全部记录为 JSON
- 支持导入 JSON，并按日期合并：同日期覆盖，新日期新增
- 使用 localStorage 保存，刷新页面或关闭网页后再打开，记录仍然保留

## 如何打开测试

直接用浏览器打开本项目里的 `index.html` 文件即可。

也可以在 Finder 中进入：

```text
/Users/jax/Desktop/Jax Ai Project/03_Learning_Tracker
```

然后双击 `index.html`。

## 数据保存在哪里

数据保存在当前浏览器的 localStorage 中，固定键名是：

```text
jax_learning_tracker_records
```

记录内部日期统一保存为 `YYYY-MM-DD` 格式，用于查找、更新、删除、导入和导出；页面显示可以是浏览器本地化的日期样式。

导出的 JSON 文件可以作为备份，也可以再导入回来恢复记录。

## 评分规则

- AI 编程：20 分
- 视频剪辑 / 素材整理：20 分
- 复盘笔记：20 分
- 认知视频：20 分
- 学习时长：最高 20 分，按 120 分钟封顶折算

最终分数最高 100 分。
